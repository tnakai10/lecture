# まとめ: 各ステップの繋がり

```
ステップ 1: HTML ファイルがそのまま返る
     ↓  「でも、ページごとにファイルを作るのは大変...」
ステップ 2: PHP ファイルも HTML を返せる
     ↓  「PHP で何ができるの？」
ステップ 3: PHP で動的にコンテンツを生成できる
     ↓  「便利だけど、全部 1 ファイルに書くと...」
ステップ 4: 1 ファイルに詰め込むと破綻する
     ↓  「じゃあどうすればいい？」
ステップ 5: 役割ごとにファイルを分ける（OOP）
     ↓  「テンプレートも PHP だと自由すぎる...」
ステップ 6: Twig テンプレートの仕組みを知る
     ↓  「実際に置き換えてみよう」
ステップ 7: PHP テンプレートを Twig に置き換える
     ↓  「全体としてどう動くの？」
ステップ 8: Drupal のリクエスト処理の全体像
```
