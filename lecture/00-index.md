## 前提知識の確認

ブラウザに URL を入力すると、以下のことが起きている。

1. ブラウザが Web サーバー（Apache や Nginx）にリクエストを送る
2. Web サーバーがファイルを探して返す
3. ブラウザが受け取った HTML を画面に描画する

この「Web サーバーがファイルを探して返す」の部分が、静的 HTML と Drupal では大きく異なる。
